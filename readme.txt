Instruktions (functions)

This is a short instructions what you can do om the app.

Calendar page:
Set time, click on calendar numbers(must be the numbers).
Set start time
Set stop time
Press "Make TimeLine"

TimeLine page:
Set start time
Set stop time
Set event name
Set event color
Press "Set event"
Press "Clear event"
Press "Start over"
Delete event when event is set
